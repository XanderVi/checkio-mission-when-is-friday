З чого мені розпочати?
Спочатку, тобі потрібно знайти який день тижню ховається за даною датою. А потім порахувати кількість днів до П'ятниці. Ця інформація буде тобі корисна "Basic date and time types".
Мені потрібна допомога з цим...
Ти можеш використати метод "date.weekday()". Повертає день тижню як ціле число, де Понеділок це 0, а Неділя - 6. 
from datetime import date

date(2002, 12, 4).weekday()
>> 2  # Середа
Я не розумію, як мені підрахувати кількість днів до П'ятниці.
Використай умову "if", щоб перевірити чи заданий день йде до П'ятниці, чи після.
Не можу зрозуміти...
Добре. 
from datetime import date

day = date(2002, 12, 4).weekday()
if day <= 4:
    ...
else:
    ...
Ясно! Але... задана дата це рядок. Як його перетворити у дату?
Як варіант можеш використати генератор списку. Ось так: 
from datetime import date

calendar = [int(i) for i in day.split('.')]  # Ми конвертуємо кожний елемент
                                             # розділеного рядку у ціле число
day = date(calendar[2], calendar[1], calendar[0]).weekday()
if day <= 4:
    ...
else:
    ...
Останнє запитання. Як фінальний повинен виглядати? 
from datetime import date

def friday(day: str) -> int:
    calendar = [int(i) for i in day.split('.')]
    day = date(calendar[2], calendar[1], calendar[0]).weekday()
    if day <= 4:
        return 4 - day
    else:
        return 11 - day

Або можна використати коротший варіант

from datetime import date

def friday(day: str) -> int:
    days = [int(i) for i in day.split('.')]
    return 4 - d if (d := date(days[2], days[1], days[0]).weekday()) <= 4 else 11 - d
